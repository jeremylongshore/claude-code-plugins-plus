# Jeremy Vertex Validator

Production readiness validator for Vertex AI, ADK, and Genkit deployments with security, compliance, and performance validation.

## Installation

```bash
/plugin install jeremy-vertex-validator@claude-code-plugins-plus
```

## Features

✅ Security validation (IAM, VPC-SC, encryption)
✅ Compliance checks (audit logging, data residency)
✅ Monitoring verification (dashboards, alerts, SLOs)
✅ Performance validation (auto-scaling, caching)
✅ Best practices enforcement

## Quick Start

"Validate my ADK agent deployment for production"

Auto-activates comprehensive validation:
- Security posture review
- Monitoring configuration check
- Performance optimization review
- Compliance audit
- Production readiness report
